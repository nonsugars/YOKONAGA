# YOKONAGA

## サービス概要
ドラッグした画像を横につなげてダウンロードできます。

## 使い方
[![YOKONAGA | 写真を横に結合するだけのシンプルサービス on Youtube](http://img.youtube.com/vi/XS9SVNNtvxA/0.jpg)](http://www.youtube.com/watch?v=XS9SVNNtvxA)

## 実装予定
1. プレビュー時に写真の追加と削除ができるように
1. 並べ方を縦や横で選べる
1. フォトフレームとか文字載せられたらいいね
